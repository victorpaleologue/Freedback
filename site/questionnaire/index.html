Developer-experience questionnaire

A multi-section survey built entirely from drop-in Freedback widgets — every answer is a W3C Web Annotation you own — with a live, in-browser results summary computed from the collected annotations.

This is not a mock. This survey publishes to a live Freedback demo server — everything you submit is public. Your answers are signed by a key generated in your browser, and you can delete them (the ×): the right to erasure, implemented. Storage is ephemeral (it may reset), and the first load after idle can take a few seconds while the server wakes.

This is the kind of multi-page survey you might build in Google Forms, Typeform, or SurveyMonkey. The difference: here each question is just a small custom element, and each answer is stored as W3C Web Annotation JSON-LD (RDF) — portable, queryable, and not locked inside a form silo. Each answer is saved as you go (the widgets publish per-interaction), so there is no single submit button. The questions below target https://survey.example/2026/devx; each question has its own #q-… fragment, so answers never cross-count.
