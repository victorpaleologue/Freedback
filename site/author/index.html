An author's feedback

Every piece of feedback in Freedback carries its author's identity IRI — at minimum, their self-signed public key. Everything with a URI deserves feedback, including that. This is deliberately understated: a place for a short note, not a public score. There's no leaderboard here.

No author id given. Click the fingerprint (#a1b2c3d4-style link) next to a piece of feedback elsewhere on the site to get here.

Identity

This is the author's own portable identity — the same key that signs everything they publish. It isn't tied to any one server: the same id would appear the same way anywhere else that indexes this author's feedback.
