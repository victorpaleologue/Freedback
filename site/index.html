Freedback

A federated, open protocol for typed feedback on anything with a URI — carried as W3C Web Annotations.

Protocol artifacts (stable URLs)

  • JSON-LD @context — /ns/context.jsonld
  • Vocabulary — /ns/freedback.ttl (rating subclasses of schema:Rating)
  • SHACL shapes — /ns/shapes.ttl (all validation lives here)

About

Anyone can attach feedback (stars, scalar, thumbs, comments, tags) to any resource, publish it to a server they choose, and have it discovered and aggregated across servers — no central gatekeeper. Feedback is signed with a portable key, so it is not locked inside the silo that collected it.

Source & docs: github.com/victorpaleologue/Freedback
