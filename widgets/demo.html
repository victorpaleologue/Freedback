Freedback widgets

Run the stack first: docker compose up --build (feedback :8080, collection :8100). These widgets read aggregates from the collection server and publish to the feedback server. Publishing needs auth — set a demo OAuth token on the feedback server (FREEDBACK_OAUTH_TOKEN/_APP/_USER) and put it in data-token, or the POST returns 401 (read-only still works).
